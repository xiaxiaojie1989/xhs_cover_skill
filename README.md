# xhs_cover_skill
根据用户上传的人像/视频截图和封面文案，生成统一风格的小红书封面。适用于 AI 营销案例、职场 AI 工具、教程类、案例拆解类内容。优先做 3:4 竖版封面，保留真实照片质感，轻美颜、强标题。Generate a consistent Xiaohongshu cover image based on the user-uploaded portrait/video screenshot and cover copy. Suitable for AI marketing case studies, workplace AI tools, tutorials, and case breakdown content.  3:4 vertical cover format.  
## 生成流程

### Step 1：判断用户意图

直接执行，不要反复确认。

- 用户说“照片换一下”：用新照片重做同一标题和风格。
- 用户说“只改文字，其他不要动”：只改文字，尽量保持原图、人像、裁剪、调色、背景和版式不变。
- 用户说“点击率低”：减少字数、放大核心钩子、提升对比度、把小字变成中等大小。
- 用户说“不要纯黑”：保留照片背景或暖色/生活化背景，不要做黑底科技海报。
- 用户说“美颜一下”：面部补光、去黑眼圈、自然唇彩、轻微磨皮，不要换脸。

### Step 2：处理文字

先在脑中完成这一步，再生成图。

1. 提取最有点击价值的关键词：AI、数字、结果、冲突、反常识。
2. 把标题拆成 3–5 行。
3. 决定高亮词。
4. 删除解释性废字。
5. 保证小红书缩略图能读清。

### Step 3：处理图片

默认处理：

- 3:4 竖版裁切。
- 人脸清晰，位置靠右或居中偏右。
- 面部打柔光。
- 去黑眼圈、轻微均匀肤色。
- 唇部自然提气色。
- 背景保留原图质感，轻微虚化。
- 不要把人物改得不像本人。

### Step 4：文字排版

优先级：

1. 大标题可读。
2. 人脸不被挡。
3. 数字和 AI 高亮。
4. 小字不小到看不清。
5. 整体不是纯黑，不像廉价广告。

文字推荐布局：

- 左上：主标题。
- 左中：第二层信息。
- 左下或底部：小字补充 / 标签。
- 右侧：人物。
